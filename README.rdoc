# YummyMark

##Information

* An application, like Delicious, that allows users to bookmark food-related URLs via email, peruse other user's food-related bookmarks and maintain a personal index of categorized foodie bookmarks

* Source code [available on Gitbub](https://github.com/alexhawkins/yummymark) 

* Demo available on [Heroku](https://yummymark.herokuapp.com)
